# Playbooks da Maria (HUB Obra 10+)

Ficheiros `.md` nesta pasta são lidos em **ordem lexicográfica** (`00_`, `01_`, …) e concatenados no arranque da app em `src/playbook_loader.py`.

- **Editar só estes ficheiros** para mudar tom, frases ou fluxos — não é preciso alterar código Python.
- **`00_router.md`** — identidade, regras globais, **tabela de roteamento** (qual doc usar).
- **`01`–`04`** — um documento por família de interlocutor/fluxo.

Para documentação de produto legada, ver também `../maria.md`.
